# Tell pip to look up torch wheels from PyTorch's CUDA-12.4 index FIRST.
# Without this line, plain `pip install` pulls the default torch wheel,
# which on recent torch is built against CUDA 13 - too new for NVIDIA
# drivers in the 5xx.xx series (like the WSL2 driver that ships with
# Windows 11 today). The cu124 index gives us a CUDA-12.4 build that
# matches `nvidia-smi`'s reported CUDA version.
--extra-index-url https://download.pytorch.org/whl/cu124
